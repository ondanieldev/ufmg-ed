Development Boards & Kits - AVR DIY Gamer Kit from Build, learn, invent, play! Kids age 10 and up can create their own video games and learn to code their inventions into reality with this build-it-yourself handheld console, which was added to MoMA's collection in 2014. It comes with open-source platform Arduino programming, a 3 x 3-inch screen, and easy-to-use software, so aspiring programmers can get inspired right away by downloading games from the Gamer library and start coding. Assembly required using soldering tool (not included). Requires one 9-volt battery (not included).
