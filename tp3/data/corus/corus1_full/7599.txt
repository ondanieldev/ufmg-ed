SIIG 6.2A USB Power Adapter 4-Port Black (AC-PW0L12-S1) SIIGs 6.2A USB Power Adapter  4-Port (Black) is a high-powered desktop charger for charging up to four USB powered devices simultaneously. It offers 4 USB ports to support charging up to 2 tablet computers and two smartphones (or other USB powered mobile devices) at the same time. It is an efficient tool to enable you to charge multiple USB devices at the same time using only one wall outlet.
